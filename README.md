# 4InARowPython
      

Created a Python command line program of a 4 in a row game. This game is made for 2 players. At each turn, a player is asked to which colmnn he wishes to insert its token. Player 1 will have a token 'X' and Player two will have a token '0'. When, the user will decide the column, the current state of the board.table will be shown. The table will be shown at every decision of the player.
